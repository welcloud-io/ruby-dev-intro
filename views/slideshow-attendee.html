HISTOIRE

- C'est un langage qui vient du Japon.

- Créé par Yukihiro Matsumoto (actuellement employé chez Heroku).

- Première version officielle (0.95) en 1995.

- A ce jour (janvier 2014) la dernière version est la 2.1.0.

PHILOSOPHIE

- Ruby est conçu pour la productivité et le plaisir du programmeur.

- Il se focalise sur les besoins humains plutôt que sur les besoins de la machine.

- Il est influencé par différents langages, en particulier Smalltalk, Lisp et Perl.

class Solution def Solution.main(table, fichiers) types_mimes = Array.new(); for i in 0..fichiers.size-1 do nomFichier = fichiers[i]; if (nomFichier.rindex(".") == nil) types_mimes.push("UNKNOWN"); else ext = nomFichier[nomFichier.rindex(".")+1, nomFichier.size-1]; ext = ext.downcase; if (table.has_key?(ext)) types_mimes.push(table[ext]); else types_mimes.push("UNKNOWN"); end end i = i + 1; end return types_mimes; end end def mime_types(table, fichiers) Solution.main(table, fichiers); end
